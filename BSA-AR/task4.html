Project Planning

Before you begin a project, it's essential to interview the client about their specific needs. Ask questions about the following topics to develop an accurate quote for the required materials (components, software, hardware, and networking) and the labor necessary to bring their vision to life. Proper scoping of the project will help you provide a precise quote.

For example, if the deployment requires 5 players or fewer, we recommend automatically provisioning players via the purple screen. You will learn how to use this method in the following exercise. However, if you need to provision many players, we offer bulk provisioning tools you can use.

Another example would be a customer who wants to use a specific partner application. It's easy to set up a network to automatically provision a player to run on a preferred CMS when the player is added to the network.

Bottom line: Upfront planning of a deployment will help you sell more players and support more customers in the long run.
