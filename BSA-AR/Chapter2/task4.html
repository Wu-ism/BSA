Update Software

Make sure your customer is running the latest player OS (Production Release) and the latest BrightAuthor:connected client. If you are using a Partner Application, check with that partner to make sure you are running the latest version of their software as well.

There are several methods you can use to update the OS of your player:

If your player is running OS 8.0.147 or above, you can update the OS when you publish a presentation to a device.

If your player is running OS 8.0.147 or above, you can also update the OS from the network tab by clicking on the cog icon next to the player, navigating to the advanced tab, selecting the version, and clicking the “Update” button.

You can update the OS when you apply a new setup package and provision a player.

For additional information on applying OS updates, see this article.

BrightAuthor:connected Client

Note: This only applies if you are using the desktop client software on Windows or MacOS. If you only use a web browser, you’ll always be using the latest version.

You can "Check for Updates" in the BA connected menu.

Now that you’ve updated the OS and the client, check to see if those steps resolved the issue you were encountering. Please note which versions of the OS and the BA connected client you installed so you can mention this in a support ticket if necessary.

You can also update to the latest client by visiting our downloads page.
