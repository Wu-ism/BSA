Do you have a Known Working Environment?

Check power, cables, network, displays, storage, and inputs/outputs. Many complex troubleshooting attempts are caused by something simple, like a faulty cable. We suggest trying these simple checks first using known good equipment and environments. Here are some example scenarios:

Example 1: The incorrect input is selected on the display.

Example 2: On an XT1144 player, there's an HDMI input and an HDMI output. The HDMI cable is plugged into the input on the player instead of the output.

Example 3: The presentation is set to run in 4K, but the display is only HD, which causes the signal to fail and no image is displayed.
