On Device Setup

After resetting a player, the resulting behavior will depend on a few different factors. If the player can reach the internet, connect to the BSN.cloud services, and there is a provisioning record, the player will try to apply the affiliated setup files automatically. If there is no device provisioning record, the player should come up to the purple activation screen and display an activation and QR code.

If the player canâ€™t connect to the internet but can connect to the local network (due to a firewall), you might see the white-background On-Device Setup screen.

In this case, you may need to work with a network administrator to create a firewall exception, manage the players on the Local Area Network without BSN.cloud Control Cloud connectivity, or simply use the player as a standalone device.
